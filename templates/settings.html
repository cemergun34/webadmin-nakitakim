REST API Güvenlik Anahtarı

nakitAkim uygulaması bu anahtarı X-API-Key header'ında göndererek REST API'yi tetikler.

API KEY {{ api_key }}
Bu anahtarı değiştirmek için config.py dosyasındaki WEBADMIN_API_KEY değerini güncelleyin veya WEBADMIN_API_KEY ortam değişkeni tanımlayın.

nakitAkim'den POST Örneği

nakitAkim projesinde Womsis verisi çekmek için aşağıdaki kodu kullanın:

import requests

response = requests.post(
    "http://localhost:5050/api/womsis/sync",
    headers={"X-API-Key": "{{ api_key }}"},
    json={
        "userid": 1,
        "start_date": "2024-01-01",
        "end_date": "2024-12-31"
    },
    timeout=60
)
data = response.json()
print(data["count"], "işlem çekildi")
